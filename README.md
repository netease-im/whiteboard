# Whiteboard
云信互动白板基于云信实时通信网络，提供完整的多人实时白板互动服务。本仓库主要展视互动白板的 SDK 和 WebView 集成示例项目，帮助开发者自由、快速构建互动白板能力。

## 示例功能
  - 支持多种白板绘制工具
  - 支持多人白板实时互动
  - 支持文档上传共享转码
